Run 'n' Strike - Game

Controls and Instructions

Move Up: Arrow Up Key

Move Down: Arrow Down Key

Shoot: Spacebar

P: Pause/Resume Game

Power-Ups:

  • Invincibility (lasts 5 seconds)
  • Speed Boost (lasts 10 seconds)

Points System:

  • Defeat Enemies: 10 points
  • Collect Coins: 5 points
